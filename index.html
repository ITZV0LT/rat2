RAT2
Warehouse Remote Management — Agent
↓ Download rat2.py

Setup Instructions

1
Open rat2.py and edit the three config lines at the top:
SECRET_KEY  MANAGER_URL  LOCATION
2
Install dependencies:
pip install websockets psutil Pillow
3
Run the agent:
python rat2.py agent
4
For silent background launch, use run_agent.vbs from the GitHub repo.
